*** WaveShop source distribution ***

REQUIREMENTS:

WaveShop is designed to be an MFC 6.0 or .NET 2005/2008 project.  It
may or may not compile correctly in other versions of VC++.

INSTALLATION:

The source distribution of WaveShop doesn't have an installer.

UNINSTALLING:

To uninstall WaveShop, you need to do two things:

1.  Delete WaveShop's program files from the folder you copied them to.
2.  Delete WaveShop's registry entries, using regedit.  The key to
delete is:

"HKEY_CURRENT_USER\Software\Anal Software\WaveShop"

NOTES:

The binary version of this release is available from the WaveShop web
site, http://waveshop.sourceforge.net/.
